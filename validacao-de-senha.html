Validar senhas usando Regex

Senha 1

Sua senha deve conter pelo menos seis caracteres, pelo menos um número e pelo menos uma letra.



Senha 2

Sua senha deve conter pelo menos 8 caracteres, pelo menos um número, pelo menos uma letra e pelo menos um caracter especial.



Senha 3

Sua senha deve conter de seis a dez caracteres, pelo menos um número, pelo menos uma letra minúscula e uma maiúscula e pelo menos um caracter especial.
